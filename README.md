Upload install-fixed.sh ke GitHub repo kamu (bisa repo baru atau existing)

Saat VPS reset, tinggal SSH masuk lalu jalankan:

curl -O https://raw.githubusercontent.com/USERNAME/NAMA-REPO/main/install-fixed.sh

chmod +x install-fixed.sh

./install-fixed.sh

Ganti USERNAME dan NAMA-REPO sesuai akun GitHub kamu.
